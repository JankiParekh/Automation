 To add 
 - git add "filename with extension"
 
 To add all file together
 - git add .
 
 To commit
 - git commit -m Enter your message
 
 To check status
 - git status
 
 To unstage
 - git rm --cached filename
 
 To check changes
 - git diff file name (if you don't give file name it will display last updated file data)
 
 To remove changes
 - git restore filename
 
 To establish connection between local git repository to remote github repository
 -  git remote add origin "link" (in our case https://github.com/JankiParekh/Automation.git)
 
 To push code from local to remote repository
 - git push origin master
